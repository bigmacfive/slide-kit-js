주요 기능

  • 아름다운 애니메이션 효과
  • 진행 표시줄 지원
  • 키보드 및 스와이프 내비게이션
  • 반응형 디자인
  • 사용자 정의 이벤트 시스템
  • 자동 재생 기능

쉬운 사용법

몇 줄의 코드만으로 멋진 프레젠테이션을 만들 수 있습니다:


// SlideKit 초기화
const presentation = new SlideKit.Presentation({
  showProgress: true,
  theme: 'default',
  animateItems: true
});

// 프레젠테이션 시작
presentation.init();

// 이벤트 리스너 등록
presentation.on('slideChanged', function(event) {
  console.log('슬라이드 변경:', event.detail);
});

SlideKit은 개발자 친화적인 API로 복잡한 프레젠테이션도 쉽게 만들 수 있습니다.
