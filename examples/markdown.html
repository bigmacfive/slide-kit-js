# 마크다운으로 작성한 SlideKit 프레젠테이션 마크다운 기반 슬라이드쇼의 예시입니다.
## 마크다운의 장점 * 작성이 쉽고 빠름 * 텍스트 기반으로 버전 관리 용이 * 다양한 서식 지원 * HTML과 함께 사용 가능
## 기본 서식 **굵게** 또는 *기울임꼴*로 텍스트를 강조할 수 있습니다. 링크도 사용 가능합니다: [SlideKit 홈페이지](#) > 인용구는 중요한 내용을 강조하는 데 유용합니다. 1. 순서 목록 1 2. 순서 목록 2 3. 순서 목록 3
## 코드 블록 마크다운에서 코드 블록을 쉽게 사용할 수 있습니다: ```javascript function createSlideshow() { const presentation = new SlideKit.Presentation({ container: '.slidekit-container', theme: 'dark' }); presentation.init(); return presentation; } ```
## 이미지 삽입 ![샘플 이미지](https://via.placeholder.com/500x300) 캡션: 마크다운에서 이미지 삽입도 간단합니다.
## 테이블 사용 | 기능 | 설명 | |------|------| | 마크다운 | 마크다운 문법 지원 | | 테마 | 다양한 테마 제공 | | 전환 효과 | 슬라이드 간 애니메이션 | | 코드 강조 | 개발자 친화적 코드 하이라이팅 |
# 슬라이드 타입 지정 이 슬라이드는 `타이틀` 타입으로 지정되었습니다.
# 전환 효과와 배경 지정 이 슬라이드는 줌 효과와 회색 배경을 사용합니다.
