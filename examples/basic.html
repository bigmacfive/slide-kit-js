SlideKit이란?

SlideKit은 웹 기반 프레젠테이션을 쉽게 만들 수 있는 자바스크립트 라이브러리입니다.

  • HTML/CSS/JS 기반 - 웹 개발자에게 친숙한 도구
  • 반응형 디자인 - 모든 화면 크기에서 잘 작동
  • 다양한 테마와 전환 효과
  • 마크다운 지원
  • 코드 구문 강조

주요 기능

  • 다양한 슬라이드 레이아웃
  • 키보드 네비게이션
  • 터치 스와이프 지원
  • 발표자 모드
  • 테마 시스템
  • 플러그인 아키텍처
# 마크다운 지원 SlideKit은 **마크다운**을 지원합니다: - 목록 항목 1 - 목록 항목 2 - 중첩 목록 - 중첩 목록 2 ```javascript // 코드 블록도 지원합니다 function hello() { console.log('Hello, SlideKit!'); } ``` > 인용구도 사용할 수 있습니다.

코드 구문 강조


import { Presentation } from 'slidekit';

// 프레젠테이션 초기화
const presentation = new SlideKit.Presentation({
  container: '.slidekit-container',
  theme: 'default',
  transition: 'fade'
});

// 프레젠테이션 시작
presentation.init();

플러그인 시스템

SlideKit은 확장 가능한 플러그인 시스템을 제공합니다:


// 간단한 플러그인 예시
const timerPlugin = SlideKit.Plugins.createPlugin({
  name: 'timer',
  init: function(presentation) {
    // 타이머 초기화
    console.log('타이머 플러그인 초기화');
  },
  destroy: function(presentation) {
    // 정리 작업
  },
  handleEvent: function(eventName, data, presentation) {
    // 이벤트 처리
    if (eventName === 'slideChanged') {
      console.log(`새 슬라이드: ${data.currentIndex}`);
    }
  }
});

// 플러그인 등록
presentation.plugins.register(timerPlugin);
